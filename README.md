# Проект Movies Explorer(frontend)  

Дипломный проект на курсе "Веб-разработчик" от Яндекс.Практикум.  

## Стек проекта  

* JAVASCRIPT  
* CRA  
* REACT ROUTING  
* HTML  
* CSS  

Запустить проект из директории проекта:  

### `npm start`  

Сделать сборку проекта.  

### `npm run build`
